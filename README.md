# ThisAbility — EquiVerse Ecosystem

**Tech For Positive Futures · Team 3 · 2026**

ThisAbility is an AI-powered disability accessibility ecosystem helping 16 million UK disabled people navigate cities independently, safely, and with dignity. This repository contains the static pitch and business website built to present the product vision, business model, and impact case.

## The product

**ThisAbility** (B2C) is a free consumer app for disabled people, powered by four modules:

- **Dynamic Map** — AI-assisted routing via Google Gemini APIs, combining live TfL fault data, OpenStreetMap, and crowd telemetry to reroute before obstacles are reached
- **CapAble ID** — Secure NFC/BLE disability profile sharing; venues receive access needs silently, no verbal explanation needed
- **Surround Alerts** — Proactive haptic warnings for micro-obstacles and sensory-heavy environments
- **Active Guardian** — 24/7 fall and tip-over detection using smartphone sensors, with instant GPS alerts to emergency contacts

**EquiVerse** (B2B) is the underlying AI and data platform — licensable to transit authorities, smart cities, NHS trusts, and employers. ThisAbility is to EquiVerse what Gemini is to Google.

## Business model

| Stream | Phase | Revenue |
|---|---|---|
| Active Guardian Premium | Phase 1–2 | £9.99/mo · target £900K ARR Year 2 |
| Corporate DEI subscriptions | Phase 2+ | Employers subsidise Active Guardian as employee benefit |
| Accessibility Heatmaps (DaaS) | Phase 3 | £50K–£100K per council/yr |
| API & integration licensing | Phase 3 | £200K–£500K per enterprise partner/yr |
| Venue listing revenue | Phase 3 | Businesses pay for accessibility endorsement badges; no user data sold |

Core navigation features are permanently free for disabled individuals. Cloud infrastructure cost: ~£32K/yr at 150K users (Gemini API inference only — no custom model training in Phase 1).
